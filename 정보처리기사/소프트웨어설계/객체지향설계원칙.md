1.객체지향설계원칙(SOLID)
=======================
### 1)Single Responsible Principle(단일 책임 원칙)
> 하나의 객체는 하나의 책임만을 지녀야 한다는 법칙
```python
# AS-IS
  class Employee:
    print('Doing Coding')
  
  def design(self):
    print('Doing Design')
  
  def analyze(self):
    print('Doing Analyze')

# TO-BE
  class Developer:
    def coding(self):
      print('Doing Coding')
  
  class Designer:
    def design(self):
      print('Doing Design')
  
  class Analyst:
    def analyze(self):
      print('Doing Analyze')
```

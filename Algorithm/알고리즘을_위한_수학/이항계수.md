* 조합의 성질인 'n = r / r = 0일 때의 조합은 항상 1이다'라는 성질을 이용하여, <b>알고리즘에서 조합 함수를 구현 가능</b>하다.


```java
public static int BC(int n, int r) {
        if(n == r || n == 0) return 1; // 'n = r / r = 0일 때의 조합은 항상 1이다'
        return BC(n - 1, r - 1) + BC(n - 1, r); // '파스칼의 법칙'
    }
```
